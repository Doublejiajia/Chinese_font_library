# Chinese font library
## 1.简介
Chinese font library用于解决嵌入式开发中需要在LCD上显示中文的问题，该软件包基于FAL软件包的功能实现，通过访问存储在外部spi flash的字库文件，实现汉字的显示。同时提供 utf-8 -> unicode -> gbk 的转换函数接口，以应用于网络数据场景。更多应用场景(例：GSM中文短信等)有待实现。
